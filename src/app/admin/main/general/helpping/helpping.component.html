How to use the featured application

When You login as an admin

You can view all tasks through the menu on the side and then click on All Tasks
You can view all users through the menu on the side and then click on All Users
You can Add a task to any user in the application through All tasks, then add a task, then click Complete. You can also Edit and Delete tasks through the menu at the end of each task in the task table
You can also follow up on tasks in case they are finished or not yet through the status in the task table
You can Change status of all users to active or inactive, or Delete any of them through the user list and then Menu
To visit your profile, click on my account from the menu on the side
